РОБОСКЛАД — ПРОТОТИП ДЛЯ ИГОРЯ

Что умеет бот:
1. Показывает остатки расходников для роботов.
2. Позволяет списывать расходники кнопками.
3. Позволяет пополнять расходники кнопками.
4. Показывает, что заканчивается.
5. Ведёт историю операций.

Расходники в демо:
- Сопла
- Наконечники
- Контактные трубки
- Проволока
- Спрей антипригарный

Запуск:

1. Открой файл .env
2. Вставь токен от BotFather:

BOT_TOKEN=твой_токен

3. Установи библиотеки:

pip install -r requirements.txt

4. Запусти:

python bot.py

5. В Telegram напиши боту:

/start

ВАЖНО:
Пока бот запущен на твоём компьютере, он работает.
Для постоянной работы нужно перенести его на VPS-сервер.
